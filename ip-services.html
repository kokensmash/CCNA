DHCP / DHCPv6

NAT & PAT

Static vs dynamic NAT, PAT, inside/outside interfaces, translation tables.

NAT1
Which two commands are used to define which interfaces participate in IPv4 NAT on a router? (Choose two.) A. ip nat inside B. ip nat outside C. ip nat enable D. ip nat source inside E. ip nat interface
Answer: A, B
NAT2
Which statement describes static NAT? A. A pool of public addresses is dynamically mapped to inside hosts. B. A single inside local address is permanently mapped to a single inside global address. C. Many inside local addresses share one inside global address using different ports. D. Inside and outside addresses are translated only when traffic is initiated from the outside.
Answer: B
NAT3
Which statement describes dynamic NAT? A. It uses a one-to-one permanent mapping. B. It maps inside local addresses to a pool of inside global addresses as sessions are created. C. It always uses the same public IP for all inside hosts. D. It translates only destination addresses.
Answer: B
NAT4

Answer: A
NAT5

Answer: B
NAT6
Which type of NAT is also known as NAT overload? A. static NAT B. dynamic NAT C. PAT D. bidirectional NAT
Answer: C
NAT7

Answer: B
NAT8

Answer: B
NAT9

Answer: B
NAT10

Answer: B
NAT11
