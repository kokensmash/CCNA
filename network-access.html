Switching Basics & MAC Table

CAM table behavior, learning/aging MACs, frame forwarding/flooding, collision/broadcast domains.

Q45

Answer: B

Q302

Spanning Tree (STP/RSTP)

Root bridge election, port roles and states, BPDU behavior, convergence.

Q62
Which two port states are part of 802.1D STP? (Choose two.) A. learning B. listening C. forwarding D. discarding E. blocking
Answer: C, E
Q68
Which statement describes RSTP topology changes? A. They require the root to send topology change notifications automatically. B. They are detected by edge ports immediately moving from blocking to forwarding. C. They are suppressed by STP BPDU filtering. D. They occur when a non-edge port transitions from discarding to forwarding or learning.
Answer: D
Q84
Which two states does an RSTP edge port immediately transition to when it receives a frame? (Choose two.) A. discarding B. learning C. listening D. forwarding E. blocking
Answer: B, D
Q101
Which two statements are characteristics of 802.1D STP? (Choose two.) A. Root ports are selected first on non-root bridges. B. It uses the concept of edge ports to speed convergence. C. It elects a single root bridge based on the lowest bridge ID. D. Alternate ports transition to forwarding when the root port fails. E. PortFast bypasses the learning state by default.
Answer: A, C
Q147
What happens to an RSTP port that receives a superior BPDU? A. It transitions to blocking immediately. B. It moves to the discarding state and resets protocol timers. C. It resigns its role and becomes the root port. D. It sends a topology change notification.
Answer: C
Q164
Which statement describes STP port states? A. The learning state allows forwarding of user traffic. B. The blocking state listens for BPDUs and does not forward frames. C. The forwarding state listens for BPDUs only. D. The shut state occurs during reconvergence.
Answer: B
Q176
A port on a non-root switch has the lowest cost path to the root bridge. What state is this port in? A. forwarding as a root port B. blocking as a designated port C. listening as an alternate port D. learning as a backup port
Answer: A
Q194
Which measure must be taken to force a switch to become the root bridge? A. Configure the root secondary command on every VLAN. B. Lower the bridge priority value. C. Configure a higher spanning-tree VLAN priority value. D. Enable BPDU guard on core-facing interfaces.
Answer: B
Q212
Which description of an RSTP edge port is true? A. A port that immediately transitions to forwarding. B. A port that must wait for the listening state before forwarding. C. A port used only between two switches. D. A port that must detect a BPDU before the edge role is assigned.
Answer: A
Q271
Which STP feature allows a port to transition directly to forwarding without waiting through STP states? A. PortFast B. BPDU Guard C. Loop Guard D. Root Guard
Answer: A
Q333
How does RSTP achieve faster convergence than STP? A. It uses bridge assurance. B. It eliminates the learning state. C. It redefines the port roles and allows rapid transition. D. It disables BPDU transmission on edge ports.
Answer: C
Q342
Which STP term corresponds to the switch port through which the root bridge is reached? A. designated port B. root port C. alternate port D. non-designated port
Answer: B
Q391
Which criterion is used to elect the root bridge in STP? A. lowest MAC address B. highest bridge priority C. lowest bridge ID D. lowest switchport cost
Answer: C
Q392
Which statement about RSTP port roles is true? A. A designated port is never used for forwarding. B. An alternate port provides an alternate path to the root bridge. C. A root port is always in blocking mode. D. A backup port replaces the designated port when the root fails.
Answer: B
Q400
Which RSTP port role provides a backup path to the root bridge? A. designated B. root C. alternate D. forwarding
Answer: C
Q403
Which port role is assigned to a port that receives superior BPDUs? A. root B. designated C. backup D. alternate
Answer: A
Q411
Which two actions does RSTP perform differently from STP? (Choose two.) A. RSTP has a blocking state. B. RSTP introduces a discarding state. C. RSTP uses alternate ports for rapid transition. D. RSTP uses inferior BPDUs to signal root changes. E. RSTP eliminates the listening state.
Answer: C, E
Q433
Which STP port state listens for BPDUs but does not forward frames? A. listening B. blocking C. forwarding D. learning
Answer: B
Q461
